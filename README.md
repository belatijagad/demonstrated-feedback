# Fork of Demonstrated Feedback

This repo is created for the purpose of recreating the experiments from Demonstrated Feedback paper by Shaikh et al.

## Usage

First, install [uv](https://docs.astral.sh/uv/) if you don't have it already:

```shell
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Then simply run the generation script:

```shell
bash run.sh
```
